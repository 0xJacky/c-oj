 # 判断星期几  
  
 ## 题目描述  
 给定y年1月1日是星期几，输入y年m月d日，计算并输出y年m月d日是星期几？  
   
 提示和要求：计算出从1月1日到m月d日的天数，利用得到的天数判定星期几。要求用switch输出星期几。  
   
 ## 输入  
 测试次数t  
   
 每组测试数据格式如下：  
   
 y/1/1 星期几(0-6，星期天到星期六）  
   
 y/m/d  
   
 （不用考虑输入非法）  
   
 ## 输出  
 对每组测试数据，输出y/m/d是星期几，具体格式见样例。  
   
 ## 样例输入  
 3  
 2017/1/1 0  
 2017/10/17  
 2000/1/1 6  
 2000/10/1  
 2012/1/1 0  
 2012/12/31  
 ## 样例输出  
 2017/10/17是星期二  
 2000/10/01是星期日  
 2012/12/31是星期一
